# Frontend CRUD Application

Frontend часть приложения, выполненная в рамках тестового задания на позицию Junior Fullstack Developer.

Проект реализован на Next.js + TypeScript + Tailwind CSS 

## Функционал

### CRUD постов
- Создание постов
- Просмотр списка постов
- Просмотр отдельного поста
- Редактирование постов
- Удаление постов

### CRUD комментариев
- Создание комментариев
- Редактирование комментариев
- Удаление комментариев
- Получение комментариев конкретного поста

### Дополнительно
- Рейтинг комментариев (1–5 звёзд)
- Валидация комментариев
- Toast уведомления через Sonner
- Загрузочные состояния
- Адаптивный интерфейс
- Компонентная архитектура
- Кастомные React hooks

### ENV

NEXT_PUBLIC_API_URL=http://localhost:3000

NEXT_PUBLIC_API_URL — адрес backend API на NestJS.
